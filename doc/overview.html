XSOAP Overview

XSOAP 1.2 is RMI system based on SOAP and interoperable with Apache SOAP (and theoretically with any other SOAP implementation).

Main features

Java version of XSOAP provides:

  • dynamic stubs using JDK 1.3+ dynamic proxies and does not require stub compiler to generate stubs or skeletons
  • adds to SOAP a simple remote reference ie. binding of endpoint URL and interface type
  • customizable XML-Java mappings and multiple SOAP serialization styles such as multi-ref or recursive, optional use of xsi:type etc.
  • support for multiple naming services through JNDI to act as RMI Registry, tested with OpenLDAP,
  • implementation of RMI Registry that is JNDI independent
  • access to web services directly without need to use naming service if necessary
  • and much much more ...
