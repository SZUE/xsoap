miniLogger provides a simple to use logging API that is very similar to Log4J and JDK 1.4 logging but works on all Java 2 versions.

To use miniLogger following steps are required:

  • add import statement, ex:

    import soaprmi.util.logging.Logger;
     
  • inside your class create instance of logger that will be named after class (ex. foo.FooClass) and will be shared by class methods, ex::

    public class FooClass {

        private static Logger logger = Logger.getLogger();

        //....

    alternatively if logger catgory should be different from automatically determined full class name ("foo.Foo") it can be set directly:

        private static Logger logger = Logger.getLogger("my.category.name");
     
  • use logger by calling logging methods on logger, ex:

     void hello(String name) {
        logger.entering(name);
        logger.fine("will do something");
        logger.exiting();
    }
